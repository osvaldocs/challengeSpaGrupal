¿Cómo funciona nuestro sistema de biblioteca Riwi?

En Riwi creemos que la colaboración entre coders es uno de nuestros pilares fundamentales. Por eso, para acceder al préstamo de libros físicos o digitales, cada usuario debe aportar al menos tres libros personales a nuestra biblioteca compartida.

Estos libros pueden ser ejemplares físicos en buen estado o eBooks en formato digital. Una vez realizado este aporte, el usuario podrá solicitar préstamos de otros títulos disponibles, fomentando así un intercambio justo y enriquecedor para toda la comunidad.
